# CrowdSense AI

## 📌 Project Overview
CrowdSense AI is an AI-powered crowd management system designed to monitor crowded places in real time using computer vision and artificial intelligence.

It helps authorities:
- Detect people
- Count crowd size
- Identify overcrowded zones
- Generate alerts
- Recommend actions
